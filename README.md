# Big Data Foundations – MovieLens Project

**Group 22 | Deadline:** 22 November 2025  
**Stack:** DuckDB (Local) | AWS S3 + Athena + Glue (Cloud)

## Goals
- Clean and analyze the MovieLens dataset locally with DuckDB
- Export cleaned data to Parquet for scalability
- Reproduce analysis in AWS Athena using the same SQL logic
- Compare performance and scalability between local and cloud environments

## Folder Structure
data/ → raw CSV files (not committed)
parquet/ → cleaned Parquet outputs
sql/ → DuckDB SQL scripts
notebooks/ → experiments (optional)
scripts/ → Python utilities
docs/ → plans, reports, and notes
